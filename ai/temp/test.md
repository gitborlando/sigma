# 编辑器测试用例

## 复制粘贴

- **操作**：选中节点，Ctrl+C，Ctrl+V
- **期望**：新增复制节点，选区切到新节点；图层无重复 key 警告

## 复制粘贴后的 Undo / Redo

- **操作**：粘贴后 undo / redo
- **期望**：undo 删除复制节点且选区不指向已删除节点；redo 恢复复制节点并选中

## 移动节点

- **操作**：选中节点拖动
- **期望**：位置变化，undo / redo 可恢复，不报 undefined matrix/type/width

## 右侧属性编辑

- **操作**：修改 x/y/width/height、fill、stroke、shadow、text 内容/样式
- **期望**：画布和右侧面板同步，undo / redo 正常

## 对齐

- **操作**：多选两个节点，点右侧对齐按钮
- **期望**：节点位置变化，undo / redo 正常

## 页面操作

- **操作**：新增页面、切换页面、删除页面
- **期望**：页面列表、舞台、选区同步；删除当前页后自动切到可用页

## 刷新恢复

- **操作**：做几步操作后刷新 mock 页面
- **期望**：mock 能重新打开，无启动错误
- **注意**：mock 不一定持久化刚才的操作，这是正常的
