请生成一个**现代专业工具型 Web 应用界面**，整体风格接近**现代 SaaS 控制台 / 开发者工具 / 专业工作台**，而不是传统企业后台或营销官网。

### 1. 总体风格

- 风格极简、克制、专业、理性
- 明显面向专业用户，而不是泛消费型用户
- 强调功能优先、信息结构清晰、主任务聚焦
- 几乎没有装饰性设计，不依赖渐变、插画、厚重阴影制造视觉效果
- 整体应呈现出现代工具软件的气质：冷静、可信、高效、可长期使用

### 2. 视觉气质

- 整体安静、低噪音、精确、有秩序
- 以白色、浅灰色、中性灰为主
- 强调“工作区”和“操作效率”，而不是“展示感”
- 界面应让人感觉这是一个可以高频使用的专业产品
- 所有视觉元素都应服务于核心任务流程

### 3. 色彩系统

- 主背景使用纯白或接近纯白
- 次级背景使用极浅灰，用于区分侧边栏、工具条、模块区、空状态区
- 主要通过 1px 浅灰边框进行分区，而不是依赖阴影
- 文本颜色以深灰和中灰为主，层级清晰但不过度强烈
- 仅在最关键操作上使用少量高饱和强调色，例如绿色或蓝色用于主按钮
- 其他控件尽量保持中性配色，避免界面过于花哨

### 4. 布局结构

采用典型的专业工具工作台布局：

- 最左侧可以有窄图标栏，用于全局导航
- 左侧可以有模块侧边栏，用于筛选、分类、资源列表、导航树或辅助操作
- 顶部是全局上下文栏，用于展示当前项目、环境、模块、搜索、帮助入口、用户菜单
- 中央是主工作区，是整个页面的视觉核心
- 底部或右侧可以有结果区、详情区、属性区、操作面板或状态面板

整个页面应体现：
**全局导航 + 模块导航 / 侧边栏 + 顶部上下文 + 核心工作区 + 辅助信息区**

### 5. 核心任务区中心化

这是最关键的一点：

- 页面视觉焦点必须落在中央主工作区
- 主工作区应留白充足、干净、无干扰
- 侧边栏和顶部导航不能抢走主注意力
- 所有布局和视觉都要围绕用户最核心的任务展开
- 不管这个产品是表单、表格、配置台、分析页、数据管理页还是内容管理页，都要保证“主任务优先”

### 6. 组件风格

组件应体现“专业工具型 UI”，而不是营销官网风格。

#### 按钮

- 主按钮使用实心强调色
- 次按钮使用 ghost 或 outline 风格
- 圆角较小，偏克制
- 按钮尺寸偏紧凑，不要过大过松

#### 输入框 / 搜索框

- 浅色边框、浅背景或白底
- 结构干净，可带小图标
- 可加入快捷键提示
- 控件高度适中偏小，符合高信息密度产品风格

#### Tab

- 样式简洁、轻量
- 当前 tab 通过文字深浅、细下划线或轻量高亮来表示
- 不要使用厚重、夸张的胶囊式设计

#### 下拉选择器 / 筛选器 / 工具栏控件

- 风格统一、低调、紧凑
- 更像工具型操作控件，而不是强表单感控件
- 强调快速扫描与快速操作

#### 面板

- 通过细边框和浅背景区分不同区域
- 不要使用厚重卡片阴影
- 更像工作台中的 pane，而不是 dashboard card

### 7. 图标系统

- 使用统一的线性图标
- 图标尺寸小，线条细，视觉重量轻
- 图标主要用于识别和导航，不承担装饰功能
- 整体应有专业软件、SaaS 控制台、开发者工具的感觉

### 8. 排版

- 主体使用现代无衬线字体
- 字号整体偏小，适合高密度信息环境
- 通过字重、颜色、间距建立层级，而不是依赖夸张字号
- 分组标题、区块标题、辅助说明应有清晰的信息层级
- 整体排版要利于扫描、理解和连续工作

### 9. 留白与密度

- 这是一个高密度但不拥挤的界面
- 控件和信息可以较多，但要通过稳定的间距体系、细边框和清晰分区保持秩序
- 不追求海报式留白，而追求精确、整洁、可扫描
- 留白应是功能型留白，服务于结构和可读性

### 10. 交互气质

- hover 效果非常轻
- 状态变化克制，不夸张
- 主操作明确，次操作后退
- 应支持高频使用场景
- 所有反馈都要清晰，但不要花哨
- 用户应感受到高效率、稳定、专业，而不是娱乐感或营销感

### 11. 适用产品类型

这种风格适合：

- SaaS 控制台
- 数据管理后台
- 配置平台
- 内容管理系统
- 审核工作台
- 运营工具
- 企业内部工具
- 开发者平台
- 分析工具
- B 端专业软件

### 12. 应避免的方向

不要做成以下风格：

- 不要像营销官网
- 不要像消费级 App
- 不要像电商后台
- 不要像 BI 大屏
- 不要使用大量彩色卡片
- 不要过度圆角、过度阴影、过度品牌化
- 不要使用强渐变、发光、玻璃拟态、毛玻璃
- 不要让页面看起来像视觉展示页，而要像真正可工作的专业工具

### 13. 一句话总结

这是一个**现代、克制、专业、以主任务区为核心的工具型 Web 应用界面**，强调白灰中性色、细边框、紧凑布局、低噪音、高结构和高效率，仅用少量强调色突出关键操作。
